# [![Typing SVG](https://readme-typing-svg.demolab.com/?font=Segoe+UI&size=23&lines=Polaris)](https://git.io/typing-svg)

Uma aplicação web de To-Do List simples, moderna e intuitiva, desenvolvida para ajudar na organização das tarefas do dia a dia. O nome Polaris reflete a ideia de ter um "norte" ou um guia visual para a sua produtividade.

## 🚀 Tecnologias Utilizadas

[![My Skills](https://skillicons.dev/icons?i=js,html,git,tailwind)](https://skillicons.dev)

## ⚙️ Funcionalidades Atuais

* Adicionar novas tarefas à lista.
* Interface amigável com design minimalista.
* *(Se já tiver, adicione: Marcar como concluída, Deletar tarefa, etc.)*

## 🔮 O que vem por aí (Futuras Implementações)

* **Persistência de Dados:** Salvar as tarefas (utilizando `Banco de Dados`) para que elas não sumam ao atualizar a página.
* **Autenticação:** Sistema de validação de usuário para acesso individual.
* **Mais Funcionalidades:** Expansão das ferramentas da lista, como edição de tarefas, separação por categorias e filtros.
